# fullpage.js

这是一个很不错的全屏插件，但是效果只有滚动的效果，我对他进行了延伸增加了渐变的效果，添加参数渐变时间。

### [使用方法 ➫](http://www.dowebok.com)

fullpagejs的用法

- JavaScript

```javascript
 $.fn.fullpage({
    verticalCentered: false,
    anchors: ['page1', 'page2', 'page3', 'page4', ],
    anamopacity: 300, //渐变的时间    这里是增加的初始参数  其他参数一样
    navigation: true,
    navigationTooltips: ['', '', '', '', '', '', '']
});
```
